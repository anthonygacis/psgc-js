# Region Code Reference

You can all also get the region code by taking the first two digits of the code. Example **"050000000"** for Bicol, the Region code will be **"05"**

| ------ | ------ |
|01 | Region I |
| 02 | Region II |
| 03 | Region III |
| 04 | Region IV-A |
| 05 | Region V |
| 06 | Region VI |
| 07 | Region VII |
| 08 | Region VIII |
| 09 | Region IX |
| 10 | Region X |
| 11 | Region XI |
| 12 | Region XII |
| 13 | NCR |
| 14 | CAR |
| 15 | ARMM |
| 16 | Region XIII |
| 17 | MIMAROPA |
